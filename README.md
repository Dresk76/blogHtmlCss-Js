# Desarrollo de Aplicaciones en Red – Laboratorios Web

## Descripción

Este proyecto fue desarrollado como parte de la asignatura **Desarrollo de Aplicaciones en Red** durante mis estudios universitarios.

La aplicación consiste en una **página web principal que organiza y presenta diferentes laboratorios prácticos**, enfocados en el uso de **HTML, CSS, JavaScript y AJAX** para el desarrollo de aplicaciones web interactivas.

El sitio permite navegar entre los diferentes laboratorios mediante un **menú lateral interactivo**, donde cada laboratorio demuestra distintos conceptos de programación web.

---

## Características

El proyecto incluye:

- Página principal con navegación hacia los laboratorios
- Menú lateral interactivo
- Interfaz responsive
- Formularios interactivos con JavaScript
- Uso de peticiones **AJAX**
- Validaciones de texto y números
- Manipulación del DOM

---

## Laboratorios incluidos

### Laboratorio 1 – Blog

Una página tipo blog desarrollada para practicar:

- Estructuración de contenido web
- Diseño con CSS
- Organización de secciones
- Navegación entre páginas

---

### Laboratorio 2 – JavaScript y AJAX

Este laboratorio incluye diferentes ejercicios interactivos desarrollados con **JavaScript**.

Funciones implementadas:

**1. Verificador de palíndromos**

Permite ingresar una palabra o frase y determinar si se trata de un **palíndromo**.

Ejemplo: Anita lava la tina


---

**2. Comparador de números**

Permite ingresar dos números y determina cuál de ellos es el **mayor**.

---

**3. Identificación de vocales**

Analiza una palabra o frase e identifica las **vocales presentes**.

---

**4. Conteo de vocales**

Cuenta cuántas veces aparece cada vocal:

- A
- E
- I
- O
- U

---

**5. Peticiones AJAX**

Permite ingresar una **URL** y realizar una petición para obtener datos desde una fuente externa utilizando **AJAX**.

La aplicación muestra:

- Estado de la petición
- Contenido recibido
- Lista de resultados obtenidos
